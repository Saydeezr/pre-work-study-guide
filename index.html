HTML

  • Want to finally know what HTML stands for? (nobody really knows what it stands for they just pretend... until now) It stands for Hypertext Markup Language! This is used for the structure of the page.
  • The head element contains information about the webpage but is not visible to the user.
  • The body element represents the visble content shown to the user.

CSS

  • This is used for the style of the webpage.
  • Can either be Inline CSS, Internal CSS sheet style, or External CSS sheet style.
  • Rules can be applied directly to a class of elements, by using the selctor such as the section class "card".
  • Contains a selector, followed by the certain property and its value that make up the declaration of that selector.

JavaScript

  • JavaScript is like learning Mandarin: difficult to the max but rewarding if you stick with it.
  • A variable is a named container that allows us to store data in our code.
  • Control Flow is the order in which a computer executes code in a script.
  • Functions are instructions that tell a computer how to do a certain task. They are the building blocks of JavaScript.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Adding the changes from git to GitHub are very important, especially when collaborating with others on a project.
  • This is composed of adding the changes to the current working branch, commiting them to be staged, pulling changes from the base branch, and pushing changes to the remote branch.
  • Dont forget to check your work and make sure you are on the correct branch!
